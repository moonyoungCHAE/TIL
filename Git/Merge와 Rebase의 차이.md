## Merger와 Rebase의 차이
* Branch를 병합시키는 방법
  1. Merge
  2. Rebase
  
<img src="https://miro.medium.com/max/855/1*pzT4KMiZDOFsMOKH-cJjfQ.png">
* Merge
  * Branch를 통합한다. 
  * 갈래가 존재한다.
* Rebase
  * 두 브랜치 간에 공통인 지점 이후부터 Branch를 만든다.
  * Base를 새로 만들 Feature Branch가 master로 부터 파생되는 지점으로 이동한다.
  * 갈래가 존재하지 않는다.
